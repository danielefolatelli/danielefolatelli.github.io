Helps calculate scores for Ticket to Ride board game.

 

With this app you can very quickly and smoothly calculate scores for the Ticket to Ride board game and all it's extensions.

You can add all points for:

 

Routes: Click on the number indicating route in train cars length. Undo button for editing. Only lengths for the extension are shown. Number of train cars left are indicated to validate that all routes were accounted for.

 

Destination Tickets: Click on the number indicating destination ticket points. With a simple click you can do the complete tickets and incomplete tickets. Undo button for editing. Only points for the actual destination tickets in the extension are shown.

 

Bonuses according to each board game extension!! All bonus points are awarded according to the rules. Each extension requires some information to be input as indicated below:

 

Longest Route: Input longest route in train cars for each player and the points are awarded to the player(s) with the longest route with the bonus corresponding to the extension.

 

Globertrotter: No need to input. Points are awarded to the player(s) with the most completed destination tickets with the bonus corresponding to the extension.

 

Europe Stations: Input how many stations were used for each player; points are awarded accordingly.

 

Rails and Sails Ports: Indicate if each port was used and how many destination tickets for that port were completed for each player; points are awarded accordingly.

 

Africa Double Routes: Input how many routes were made with terrain cards for each player; extra points are awarded accordingly. For this expansion it is necessary to simply take note of routes made with terrain cards for each player during game play.

 

India Mandalas: Input how many mandalas for each player; points are awarded accordingly.

 

Asia Mountain: Input how many train cars were used for mountains for each player; points are awarded accordingly.

 

Nederland Toll Tokens and Loans: Input how many loans were taken and toll tokens at the end of the game for each player; points are awarded accordingly.

 

Pennsylvania Shares: For each player input how many cards and best card for each share; points are awarded accordingly. For two player games input the dummy player information with the half of the cards that are used for scoring.

 

United Kingdom Cards: For each player indicate the cards that were bought; points are awarded accordingly. Only cards that award points are considered. For the Steam Turbine card input number of ferries built at the time the card is bought and ferries at the end. For the Double Heading card input number of routes built at the time the card is bought. For this expansion it is necessary to simply take note of this information during game play.

Germany Passengers: Input number of meeple passengers at end game for each player; points are awarded accordingly. Ties are considered for scoring as indicated in the rules.

 

Old West Cities: This extension requires input during game play. Each time a route is built and there are cities take note of points for cities. Points are awarded for player(s) that owns the city and subtracted for the player that built the route so they are subtracted from the regular way of inputting route points.

 

 

 

The app is intended to be used as the game finishes and all points are counted for.
