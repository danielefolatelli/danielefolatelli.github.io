The Wonders 7 app allows you to calculate scores for the 7 Wonders board game. With the app you can input all the information at the end of the game and know who won.




When you finish a game you will be shown if any records were broken or presonal best scored. Scores are saved, so you can view all the history.




You can input the scores with just a few clicks. The format is very similar to the scorecard included in the game. But you have different input screens for each score. This is just an example of the science input screen.




Install the Wonders 7 app directly from the Google Play Store or Apple App Store!
